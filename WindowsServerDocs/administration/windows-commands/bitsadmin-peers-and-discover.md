# <a name="bitsadmin-peers-and-discover"></a>bitsadmin peers e discover

Descobre os pares novamente.

## <a name="syntax"></a>Sintaxe

```
bitsadmin /peers /discover
```

## <a name="examples"></a>Exemplos

Para descobrir os pares novamente:

```
bitsadmin /peers /discover
```
