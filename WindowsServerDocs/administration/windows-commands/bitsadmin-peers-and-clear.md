# <a name="bitsadmin-peers-and-clear"></a>bitsadmin peers e clear

Limpa a lista de pares.

## <a name="syntax"></a>Sintaxe

```
bitsadmin /peers /clear
```

## <a name="examples"></a>Exemplos

Para limpar a lista de pares.

```
bitsadmin /peers /clear
```
